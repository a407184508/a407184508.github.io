- ccleaner: 免费清理工具
- google-chrome: chrome 
- flutter 
- utools: 工具箱
- visual-studio-code
- betterzip: rar 压缩工具
- font-fira-code: 字体配置
- macdown: markdown 文档
- iterm2: 终端

其他环境配置:
- node
- flutter 
- python
- java
