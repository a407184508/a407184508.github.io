
# django 学习笔记

## 1. 简介

Django 是一个高性能的 Web 应用框架，用于开发大型 Web 应用。

Django 是一个开放源代码的 Web 应用框架，由 Python 编写。

Django 是一个基于 MVC 框架的 Web 应用框架，由 Python 编写。

Django 是一个免费的 Web 应用框架，用于开发大型 Web 应用。

Django 是一个开放源码的 Web 应用框架，用于开发大型 Web 应用。

Django 是一个基于 MVC 框架的 Web 应用框架，用于开发大型 Web 应用。

## 2. 安装

```bash
pip install django
```

## 3. 项目结构

```bash
├── manage.py
├── mysite
│   ├── __init__.py
│   ├── asgi.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── mysite.settings.py
└── mysite.urls.py
```

## 4. 运行

```bash
python manage.py runserver
```
