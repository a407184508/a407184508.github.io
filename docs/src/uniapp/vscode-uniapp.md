
# 在 vscode 中配置 uniapp

## 配置uniapp环境

使用命令行创建项目

1. 安装uniapp插件
	a. 快速创建页面 `uni-carete-view`
	b. uniapp代码提示 `uni-helper`
	c. 鼠标悬停查看文档 `uniapp小程序扩展`

2. TS 类型校验
	a. 安装类型声明文件
	b. 配置 tscofing.json
	
3. json 注释问题
 - 在设置里查找`文件关联` manifest.json 和 pages.json 添加成 jsonc 文件类型
